## How to run the app

- `npm install` at the root of the project to download the dependencies.
- `npm run dev` in another terminal to launch the application.

## How to run mock API

- `npm run api` to run the mock backend (json-server).

## Notes

MUI version ^6 was chosen for this submission. To that end, I used Next.js version ^15 as latest stable (version ^16) is not supported by MUI version ^6.
